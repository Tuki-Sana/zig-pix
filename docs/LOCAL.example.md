# ローカルメモ（`docs/LOCAL.md` 用テンプレ）

**このファイルはコミットする。** 個人用の内容は **`docs/LOCAL.md`** に書く（`.gitignore` 対象でリポジトリに上がらない）。

```bash
cp docs/LOCAL.example.md docs/LOCAL.md
# `docs/LOCAL.md` を編集する
```

## リポジトリの Markdown に書かないほうがよいもの（チェック用）

- 認証情報（npm トークン、API キー、`.npmrc` の秘密行 等）
- 社内限定 URL・ホスト名・個人連絡先（公開リポに不要なもの）
- 未整理のセキュリティ詳細（GitHub の Security 運用に従う）
- 他人の個人情報・顧客データ・契約上出せない内容

## 自分用メモ（自由記述）

- 例: 直近の `RUN_ID`、publish 前の確認、社内承認のリンク 等
