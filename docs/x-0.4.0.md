# X 投稿案（スレッド形式）

---

**1/5**

Zig 製の画像処理ライブラリ「zenpix」v0.4.0 を npm に公開しました。

Node.js / Bun / Deno すべてに対応。`npm install zenpix` だけで動きます（外部ライブラリの追加インストール不要）。

https://www.npmjs.com/package/zenpix

---

**2/5**

v0.4.0 で API が一通り揃いました。

```ts
import { decode, resize, encodeWebP, encodeAvif, encodePng, crop } from "zenpix";

const image = decode(readFileSync("photo.jpg")); // JPEG/PNG/WebP
const resized = resize(image, { width: 1920 });  // Lanczos-3
const avif = encodeAvif(resized, { quality: 60, speed: 10 });
const thumb = crop(resized, { left: 0, top: 0, width: 400, height: 300 });
```

---

**3/5**

今回のハイライトは **EXIF Orientation 自動回転**。

スマホ撮影の JPEG を `decode()` に渡すだけで、横倒し・逆さまを自動補正して返します。Orientation 1〜8 全対応。width / height も正位置のサイズで返ってきます。

---

**4/5**

ベンチマーク（少コア VPS: Ubuntu vCPU 2 / RAM 2 GB）
パイプライン: PNG デコード → リサイズ → AVIF エンコード

キャラ・厚塗り系で **Sharp の 1.2〜1.5× 速い** 結果でした（0.4.0 バイナリ、3 回計測の中央値）。シングルスレッドで動くため CPU user 時間も少なく、サーバの CPU 予算が気になる場面でよく使っています。

※ マルチコア・高スペック Mac では Sharp が速いです。用途に合わせて使い分けを。

---

**5/5**
